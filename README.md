Simulador de e-commerce para una bodega de vinos, o tienda online de bebidas alcoholicas.

Muestra los productos almacenados en firebase a modo de catalogo de bebidas, permitiendo filtrar por categoria y acceder al detalle de cada producto. Una vez en el detalle del producto, permite determinar una cantidad de unidades y agregarlas, mientras haya stock, al carrito de compras. Al acceder al carrito, se visualiza una lista con todos los productos cargados, permitiendo vaciar el carro, sumar o restar unidades de los productos, y finalizar la compra.
Para finalizar la compra y obtener el ID de la operacion, se debe rellanar un formulario con los datos del usuario y validar el email.
